## 1. Миграции 
Миграции - это система контроля версий для структуры базы данных Они позволяют придерживаться единой структуры БД в командной разработке.
Миграции дают возможность создавать, изменять и удалять таблицы с помощью PHP-кода, избавляя от необходимости писать SQL-запросы вручную.

## 2. Eloquent ORM 
'Eloquent ORM' нужна для того, чтобы связывать таблицы базы данных с объектами в коде (моделями), позволяя управлять данными с помощью элегантного объектно-ориентированного синтаксиса вместо написания сложных SQL-запросов.
Eloquent ORM обеспечивает:
1. удобное управление связями (Relationships) с помощью методов: hasOne, hasMany, belongsToMany;
2. безопасность данных (защищата от SQL-инъекций).
3. автоматизацию - создает и обновляет поля created_at и updated_at
4. возможность написание кода для работы с разными базами данных (MySQL, SQLite, PostgreSQL... )
